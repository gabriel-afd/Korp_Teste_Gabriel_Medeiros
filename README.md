# KorpERP

Repositório com dois projetos principais:

- **Back-end:** `back-end/`
- **Front-end:** `front-end/`

---

## Resumo rápido

- Abra o back-end no Visual Studio: `back-end\KorpERP\KorpERP.sln`
- Abra o front-end no VS Code ou outro editor: `front-end/`

---

## Pré-requisitos

- .NET 8 SDK instalado
- Para front-end: Node.js + npm/yarn (se aplicável)

---

## Comandos úteis (PowerShell)

### Back-end

```powershell
cd F:\KorpERP\back-end\KorpERP
dotnet build
dotnet run --project .\KorpERP.csproj
```

> Ou abra `KorpERP.sln` diretamente no Visual Studio.

### Front-end

```powershell
cd F:\KorpERP\front-end
npm install
npm start
```

> Ou use `yarn start`, conforme o projeto.

---

## Observações

- Cada pasta (`back-end` e `front-end`) contém um `README.md` com instruções específicas do respectivo projeto.
- Este arquivo serve como ponto de entrada para navegar pelo repositório. Ajuste os comandos conforme sua configuração local (nomes de projetos, portas, variáveis de ambiente).
